	<data name="HowToPlayText" xml:space="preserve">
		<value>To create an object (planet, star...), just touch for the point where it will appear (do not release the finger), drag for drawing the force vector (initial speed and direction) and release the finger to create your astronomical object. The longer is the line you draw, the higher will be the speed of the new object.

You can change the mass of your next object using the left control slider. Heavier objects, due to their higher gravity, will affect more other orbits so the system stabilization will become difficult.

When two objects impact, the heavier will absorb the lighter. 

When an object crosses the outer red frontier, it is lost from the system.

You can enter in camera mode by clicking the second button (camera) on the sidebar. Now you will be able to move the camera and zoom. Your position and the width of the space in the screen are expressed in astronomical units (AU). Use the left slider if you want a faster zooming. Pinch with two fingers for more accurate zooming.

In this mode you can touch any object to track it. You will also see the barycenter of the system, wich is also trackable.

While following an object, you can create a planet (by going back to the objects button, the first one). Its initial movement will be relative to the followed object. This is useful for creating moons, for example. 

There are different game modes with different goals, where you can score and stablish your own personal record. In Sandbox mode you can't get points, use it for training and experiment with your own solar systems without restrictions.

Each map has its own characteristics like maximum and minimum mass for creating objects, number of objects, zoom and size (playable area).

In addition to use AU for distances, one second in the game equals one year. Einstein's relativity is not considered so faster-than-light objects are possible.

It is also possible to pause and share a screenshot of your system.

Have fun :)</value>
